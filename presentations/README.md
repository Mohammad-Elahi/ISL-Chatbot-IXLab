# Sample Presentation

This is a sample presentation you can use as a template for your own projects.
Clone it, create a new repository with your group id and prepare your presentation.

To set it up quick and test (you might have nodeJs Installed):
```
npm install -g reveal-md
reveal-md sample.md --template style/settings.html
```


## setup using reveal-md

We use [reveal-md](https://github.com/webpro/reveal-md), a markdown optimized version of [reveal.js](https://revealjs.com/).
That means all presentations are basically markdown files, taht will be rendered by the reveal-md engine.

To get started, you need to:

**install nodeJs** : [https://nodejs.org/de]

**install reveal-md** : 
```npm install -g reveal-md```

## show the slides
* Use a command line interface (bash, cmd, git bash) and enter the prsentation folder
* enter ```reveal-md name-of-your-slide.md --template style/settings.html```, this will open your browser with the presentation
